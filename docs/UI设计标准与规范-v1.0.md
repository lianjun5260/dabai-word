## 1. 设计原则与理念

### 1.1 核心设计理念

**"大白陪伴，科学高效"**

以 IP 角色「大白」贯穿全应用体验，传递温暖、可靠、智能的品牌形象。视觉风格追求：

- **简洁克制**：减少装饰性元素，让内容和学习本身成为焦点
- **正向激励**：通过色彩、动画、文案传递积极情绪
- **清晰层级**：信息层次分明，用户一眼找到关键操作

### 1.2 设计关键词

| 关键词 | 说明 |
|--------|------|
| 温暖 | 绿色主色调 + 圆角卡片 + 大白IP形象，传达亲和力 |
| 清晰 | 高对比度文字、明确的状态色、一致的交互反馈 |
| 高效 | 操作路径最短化，核心功能 ≤ 2 步触达 |
| 激励 | 答对即时正反馈，完成学习有庆祝动效 |

## 3. 字体排版

### 3.1 字体族

| 平台 | 字体 |
|------|------|
| iOS | -apple-system, BlinkMacSystemFont, "SF Pro Text", "PingFang SC" |
| Android | Roboto, "Noto Sans SC" |
| 微信小程序 | -apple-system, BlinkMacSystemFont, "PingFang SC", "Helvetica Neue", sans-serif |
| 鸿蒙 | HarmonyOS Sans |

> **英文单词展示统一使用系统默认无衬线字体（San Francisco / Roboto），不指定特殊英文字体。**

### 3.2 字号阶梯

| 层级 | 字号 | 字重 | 行高 | 用途示例 |
|------|------|------|------|---------|
| H0 - 大标题 | 22px | Bold (700) | 1.3 | 庆祝页大字（如 "91分"） |
| H1 - 页面标题 | 18px | Bold (600) | 1.4 | 导航栏标题、页面主标题 |
| H2 - 区块标题 | 16px | Semibold (600) | 1.4 | 卡片内区块标题（"当前在学"、"今日计划"） |
| H3 - 单词展示 | 28~32px | Regular (400) | 1.2 | 学习组件中的英文单词 |
| Body - 正文 | 15px | Regular (400) | 1.6 | 选项文字、列表项、按钮文字 |
| Body - 小 | 14px | Regular (400) | 1.6 | 辅助说明、释义文字、例句翻译 |
| Caption - 说明 | 12px | Regular (400) | 1.5 | 提示文字（"上滑下一词"、"请选择..."）、时间戳 |
| Caption - 迷你 | 10px | Regular (400) | 1.4 | 标签内小字（"adj."、"n."）、角标文字 |

### 3.3 字重使用规则

| 字重 | 使用场景 |
|------|---------|
| **Bold (700)** | 数字统计（如 "20/20词"）、页面标题、庆祝页主文字 |
| **Semibold (600)** | 区块标题、卡片标题、按钮文字 |
| **Medium (500)** | 选项中被选中的文字、强调信息 |
| **Regular (400)** | 默认正文、单词展示、输入框文字 |

### 3.4 特殊排版规则

**英文单词展示**：
- 字号：28~32px（根据单词长度自适应）
- 字重：Regular（不加粗）
- 字母间距：正常（letter-spacing: 0）
- 大写：全部小写（除非专有名词）

**音标展示**：
- 字号：14px
- 颜色：#8C8C8C
- 格式：方括号包裹，如 `[ˈɪndəstriəs]`
- 英式/美式切换图标在音标左侧

**中文释义**：
- 字号：15px
- 颜色：#262626
- 多条释义用"；"分隔

## 4. 间距系统

基于 **4pt 基础网格**（即所有间距为 4 的倍数）。

### 4.1 全局间距常量

| 名称 | 值 | 用途 |
|------|-----|------|
| xs | 4px | 图标与文字间距、标签内 padding |
| sm | 8px | 行内元素间距、紧凑列表项间距 |
| md | 12px | 列表项间距、卡片内元素间距 |
| lg | 16px | 页面水平 padding、卡片内边距 |
| xl | 24px | 区块之间的大间距 |
| xxl | 32px | 页面顶部安全区补充、大区块间距 |

### 4.2 各场景间距细则

**页面结构**：

```
┌─────────────────────────┐
│  状态栏（系统自动）       │  ← 安全区域
├─────────────────────────┤
│  导航栏（高度 44px）      │  ← 左箭头(16) + 标题居中 + 右侧图标(16)
│  padding: 0 16px        │
├─────────────────────────┤
│                         │
│  内容区域                │  ← padding: 16px 水平
│  padding: 16px          │     子元素间距: 12px
│                         │
├─────────────────────────┤
│  底部操作区              │  ← 固定底部，padding: 12px 16px
│  (录音按钮/主操作按钮)    │
├─────────────────────────┤
│  底部导航栏（高度 56px）   │  ← Tab Bar
│  安全区（34px iPhone X+） │
└─────────────────────────┘
```

**卡片内部间距**：

| 元素 | 间距 |
|------|------|
| 卡片外边距 | 上下 12px，左右 16px（或左右 0 如果全宽） |
| 卡片内边距 | 16px |
| 卡片圆角 | 12px |
| 卡片间距离 | 12px |
| 标题与内容间距 | 12px |
| 内容行高 | 1.5~1.6 |

**列表项间距**：

| 元素 | 间距 |
|------|------|
| 列表项高度 | 最小 52px（含内边距） |
| 列表项内边距 | 16px 水平，12px 垂直 |
| 列表项间距 | 无（由分割线或背景区分） |
| 分割线高度 | 0.5px 或 1px，颜色 #F0F0F0 |

**按钮尺寸**：

| 按钮 | 高度 | 内边距 | 圆角 |
|------|------|--------|------|
| 主按钮（大） | 48px | 0 24px | 8px (pill) 或 6px |
| 次按钮（中） | 40px | 0 20px | 6px |
| 小按钮/标签 | 28px | 0 12px | 14px (pill) |
| 图标按钮 | 36px | — | — |
| 选项卡（单选） | 48px | 0 16px | 8px |

## 6. 图标规范

### 6.1 图标风格

- **线性图标**（Line Icon）：大部分场景使用 1.5px 或 2px 描边的线性图标
- **填充图标**（Filled Icon）：底部导航栏选中态、状态指示等少量场景
- **图标尺寸**：
  - 导航栏图标：20×20px
  - 功能入口图标：24×24px
  - 底部导航图标：24×24px
  - 工具栏图标：20×20px（原声/收藏/发音）

### 6.2 常用图标清单

| 图标 | 名称 | 使用场景 |
|------|------|---------|
| `<` | 返回箭头 | 导航栏返回按钮 |
| `...` | 更多菜单 | 学习组件右上角菜单 |
| `○` | 小程序胶囊 | 右上角固定位置 |
| 🔊 (扬声器) | 发音/播放 | 单词发音按钮、例句播放 |
| ⭐ (星形) | 收藏/评分 | 收藏按钮、星级评分 |
| ✓ | 对勾 | 正确标记、选中状态 |
| ✕ | 叉号 | 错误标记、关闭按钮 |
| 🏠 | 首页 | 底部导航 |
| 📊 | 统计 | 底部导航 |
| 👤 | 我的 | 底部导航 |
| > | 箭头 | 列表项右箭头、详情入口 |
| ⟲ | 刷新/切换 | 年级切换、换一批 |
| ☐ / ✓ | 复选框 | 设置学习目标的多选项 |
| 🔒 | 锁定 | VIP 功能锁定标识 |

### 6.3 大白 IP 形象

大白是本应用的伴学 IP 角色，出现在以下位置：

| 位置 | 规格 | 说明 |
|------|------|------|
| 登录页中央 | 120×120px，圆形绿色底 | 完整正面形象 |
| 首页分享横幅 | 80×80px | 半身形象 + 文案 |
| 新学/复习完成弹窗 | 可选出现 | 庆祝氛围 |
| 加载状态 | 可选出现 | 等待时的趣味元素 |

**大白形象特征**（供设计师参考）：
- 白色圆润机器人外形，类似 Baymax（超能陆战队）
- 简洁黑色眼睛（两个椭圆点）
- 温和友好的表情
- 配合绿色品牌色调的背景

## 7. 通用组件库

### 7.1 导航栏 (Navigation Bar)

**结构**：`[< 返回] [页面标题居中] [... ○]`

| 属性 | 规格 |
|------|------|
| 高度 | 44px（不含状态栏） |
| 背景 | #FFFFFF 或透明（滚动时变白） |
| 标题字号 | 18px，Semibold (600)，#262626 |
| 返回图标 | < ，20×20px，#262626，左边距 16px |
| 右侧图标 | ... 和 ○，间距 16px，右边距 16px |
| 底部分割线 | 0.5px，#F0F0F0 |

**变体**：
- 学习组件导航栏：额外显示进度文字（如 "今日进度 1/10"）
- 首页/我的页面：无返回按钮，显示应用 Logo/名称作为左标题

### 7.2 底部导航栏 (Tab Bar)

| 属性 | 规格 |
|------|------|
| 高度 | 56px（iPhone X+ 含安全区 34px） |
| 背景 | #FFFFFF |
| 上分割线 | 0.5px，#F0F0F0 |
| Tab 数量 | 3 个（首页 / 统计 / 我的） |
| Tab 宽度 | 均分（33.3% 每个） |
| 图标尺寸 | 24×24px |
| 文字字号 | 10px |
| 未选中色 | #8C8C8C（图标 + 文字同色） |
| 选中色 | #52C41A（图标 + 文字同色） |
| 图文间距 | 4px |

**Tab 定义**：

| Tab | 图标 | 文字 | 目标页面 |
|-----|------|------|---------|
| 首页 | 🏠（房子轮廓） | 首页 | Home |
| 统计 | 📊（柱状图） | 统计 | Stats |
| 我的 | 👤（人形） | 我的 | Mine |

```
│                         ✓  │  height: 52px (min)
└─────────────────────────────┘
```

**状态变体**：

| 状态 | 背景色 | 边框 | 文字 | 右侧图标 |
|------|--------|------|------|---------|

**选项内部结构**：
- 第一行：选项主文字（15px，#262626）
- 第二行（可选）：辅助文字（12px，#8C8C8C），如 "adj.勤奋的"

### 8.4 录音按钮规范

| 属性 | 值 |
|------|-----|
| 尺寸 | 宽度 100%（减去水平 padding），高度 48px |
| 背景色 | #52C41A |

| 圆角 | 6px |
| 默认态 | 绿色底白字 |
| 按压态 | 深绿底(#389E0D)，显示绿色波形动画 |

**录音状态流转图**：

```
默认态 → [长按] → 录音中(波形+倒计时)

                    │                      │            ├─→ [点击查看评价] → 评价弹窗

                    │                      └─→ 超时 → 错误Toast("分析超时") → 回到默认态
                    └─→ [上滑] → 取消录音 → 回到默认态
```

## 11. 动效规范

### 11.1 动效原则

- **快速响应**：所有交互动效 ≤ 300ms
- **自然流畅**：使用 ease-out 曲线（减速）
- **不干扰**：动效服务于功能，不为炫技
- **尊重设置**：遵循系统的「减弱动态效果」开关

### 11.2 交互动效清单

| 动效 | 触发时机 | 时长 | 类型 |
|------|---------|------|------|
| 按钮按压缩小 | 点击主按钮 | 100ms | scale(0.98) |
| 选项选中高亮 | 点击选项 | 150ms | 背景色过渡 |
| 答对打勾动画 | 答对时 | 300ms | ✓ 从左滑入 + 缩放 |
| 答错抖动 | 答错时 | 400ms | 左右 shake |
| 录音波形脉冲 | 长按录音中 | 循环 | 波形起伏 |
| 评分星星依次点亮 | 评测结果 | 500ms | 星星逐个放大 |
| 庆祝彩纸飘落 | 学习完成 | 2000ms | 粒子从顶部落下 |
| 弹窗弹出 | 模态框出现 | 250ms | 从下往上 slideUp + fade |
| 弹窗关闭 | 点击关闭/背景 | 200ms | fadeOut |
| 页面转场 | 页面切换 | 250~300ms | 平台默认（小程序原生转场） |
| 进度条增长 | 学习进度更新 | 300ms | width 过渡 |
| Toast 出现 | 提示消息 | 200ms | fadeIn + slideUp |
| Toast 消失 | 自动 | 200ms | fadeOut |
| 下一个词滑入 | 切换单词 | 250ms | 从右往左 slide |
| 单词展开/收起 | 点击查看词义 | 200ms | height 展开/收起 |

### 11.3 动效曲线

| 场景 | 曲线 | CSS 值 |
|------|------|---------|
| 进入/出现 | ease-out | `cubic-bezier(0.0, 0.0, 0.2, 1)` |
| 退出/消失 | ease-in | `cubic-bezier(0.4, 0.0, 1, 1)` |
| 弹性反馈 | spring | `cubic-bezier(0.175, 0.885, 0.32, 1.275)` |
| 线性匀速 | linear | `linear` |

## 12. 适配规范

### 12.1 屏幕尺寸适配

| 设备 | 逻辑宽度 | 设计基准 | 备注 |
|------|---------|---------|------|
| iPhone SE | 375pt | 以此为最小基准 | 确保小屏可用 |
| iPhone 14 Pro Max | 430pt | 最大宽度参考 | 内容区适当扩展 |
| Android 主流 | 360~412dp | 居中适配 | dp 与 pt 近似 1:1 |
| 小程序 | 750rpx | rpx = 2 × px | 设计稿 2x 出图 |

### 12.2 安全区域

| 平台 | 顶部 | 底部 |
|------|------|------|
| iPhone X+ (刘海) | 状态栏 44px + 导航栏 | Tab Bar 56px + Home Indicator 34px |
| iPhone 8 及以下 | 状态栏 20px | Tab Bar 49px |
| Android | 状态栏 ~24px（异形屏更高） | Navigation bar + Home Indicator |
| 小程序 | 胶囊按钮位置避让 | 底部安全区自动处理 |
| 鸿蒙 | 系统避让区 | 系统避让区 |

### 12.3 字号适配

基础字号以 **375pt（iPhone SE）** 为基准。在更大屏幕上：
- 标题/单词字号：不变
- 内边距/间距：不变
- 卡片最大宽度：限制在 680px 以内（iPad 上居中显示，两侧留白）

### 12.4 深色模式（Dark Mode）

> **首期不支持深色模式**。后续版本若支持，需额外定义暗色配色方案。预留能力：所有颜色变量已提取为 CSS 变量/主题 token，切换时批量替换即可。
